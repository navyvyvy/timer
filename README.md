# Step Timer

로스팅, 만두, 라면, 계란처럼 단계가 있는 작업을 놓치지 않도록 만든 정적 웹 타이머입니다. 각 타이머는 따로 관리되고, 카운트다운, 단계 알람, 연장, 같은 리스트 다시 시작까지 한 화면에서 처리합니다. 프리셋 패널은 로스팅, 라면, 만두, 계란에서 볼 수 있습니다.

## 빠른 시작

브라우저에서 바로 열 수 있습니다.

```text
index.html
```

홈 화면 추가와 오프라인 캐시까지 확인하려면 간단한 로컬 서버로 여는 게 좋습니다.

```bash
python -m http.server 8000
```

```text
http://localhost:8000/index.html
```

## 핵심 흐름

### 타이머 프로필

- 상단에서 `로스팅`, `만두`, `라면`, `계란`을 전환할 수 있습니다
- 진행 중에는 모드 전환과 광고 영역을 숨겨 클릭 실수를 줄입니다
- 각 프로필은 별도 저장 키로 예열 시간, 목표 온도, 단계 구성을 보존합니다
- 기존 로스팅 저장값은 새 로스팅 프로필에서도 그대로 이어서 읽습니다
- 로스팅은 원두 프리셋과 예열, 투입 온도, 단계 시간을 함께 보여줍니다

### 로스팅 프리셋

- 커피시스 원두를 중심으로 브라질, 콜롬비아, 에티오피아, 케냐, 페루, 과테말라, 온두라스, 니카라과, 엘살바도르, 동티모르, 인도, 인도네시아, 자메이카, 예멘, 탄자니아, 르완다, 파나마, 하와이까지 50여 종 이상을 담았습니다
- 약배전, 중배전, 강배전과 아프리카, 중남미, 아시아, 워시드, 내추럴, 디카페인 필터를 제공합니다
- 가나다순 정렬과 추천순 정렬, 텍스트 검색필터를 함께 제공하고, 선택하면 예열 시간, 목표 온도, 로스팅 단계 시간, 원산지, 종류, 가공, 향미 노트가 함께 바뀝니다
- 추천순은 국내에서 익숙한 원두를 앞에 두고, 뒤로 갈수록 해외의 클래식한 품종과 개성 있는 원두를 더 넓게 보여줍니다
- 즐겨찾기 별표와 즐겨찾기만 보기 필터를 제공합니다
- 설명 블록은 높이를 고정해 두고, 길어지면 내부에서만 스크롤되게 했습니다
- 선택한 로스팅 프리셋과 필터 상태는 브라우저에 저장됩니다

### 라면 제품 프리셋

- 라면 프로필에서는 유명 봉지라면 프리셋을 제공하고, 처음에는 인기순으로 보여줍니다
- `회사별`, `끓이는 라면`, `비비는 라면`, `인기순 / 가나다순` 필터를 유지합니다
- 텍스트 검색필터로 제품명을 빠르게 찾을 수 있습니다
- 즐겨찾기 별표와 즐겨찾기만 보기 필터를 제공합니다
- 제품을 선택하면 1봉 기준, 총 조리 시간, 조리법, 설명, 출처 링크가 표시됩니다
- 끓이는 라면과 비비는 라면은 각 제품의 첫 조리 단계를 기준으로 타이머를 맞춥니다
- 사용자가 시간을 조절한 뒤에도 `기본값 적용`으로 선택한 제품 기준을 다시 불러올 수 있습니다
- 선택한 라면 제품, 필터 상태, 즐겨찾기, 조정한 마일스톤은 브라우저에 저장됩니다

### 만두 제품 프리셋

- 만두 프로필에서는 국내 냉동만두와 동원 딤섬 프리셋을 제공합니다
- `냉동 / 해동` 기준을 전환할 수 있고, 회사별 필터와 인기순 / 가나다순 정렬도 유지됩니다
- 텍스트 검색필터로 제품명을 빠르게 찾을 수 있습니다
- 즐겨찾기 별표와 즐겨찾기만 보기 필터를 제공합니다
- 제품을 선택하면 찜기 기준 설명, 총 시간, 출처 링크, 기본 마일스톤이 표시됩니다
- 동원 딤섬 샤오롱바오, 새우하가우, 부추창펀, 부채교를 포함합니다
- 선택한 만두 제품, 필터 상태, 즐겨찾기, 냉동 / 해동 기준은 브라우저에 저장됩니다

### 계란 타이머

- 계란 프로필에서는 반숙, 중간반숙, 완숙 프리셋을 제공합니다
- 냉장 / 상온 기준을 전환할 수 있고, 선택한 상태에 맞는 기본 시간을 바로 보여줍니다
- 끓는 물부터 시작하는 보편적인 삶기 시간을 보여줍니다
- 선택한 계란 프리셋과 상태는 브라우저에 저장됩니다

### 예열 타이머

- 로스팅 프로필에서만 보입니다
- 선택한 로스팅 프리셋의 예열 시간과 목표 온도를 바로 보여줍니다
- 실행 중 시간 보정: `-1분`, `-30초`, `-10초`, `+10초`, `+30초`, `+1분`
- 완료 시 화면 점멸, 소리, 진동 알람

### 집중 화면

- 시작 전에는 총 시간을 크게 보여줍니다
- 시작 후에는 중앙 숫자를 `다음 행동까지` 남은 시간으로 바꿉니다
- 총 시간은 별도 참조 줄에 계속 표시합니다
- 진행 중에는 현재 마일스톤만 강조하고 다른 단계는 숨깁니다
- 현재 마일스톤의 기준 시간과 보정 버튼은 계속 보여줍니다
- 완료 후에는 이번 타이머의 실제 시간을 중앙에 그대로 보여줍니다

### 마일스톤 관리

- 로스팅 기본 단계: `건조`, `갈변`, `개발`
- 만두 기본 단계: 선택한 제품 프리셋의 찜기 기준 냉동 / 해동 조리
- 라면 기본 단계: 선택한 제품 프리셋의 조리 기준
- 각 단계는 독립 구간 시간으로 계산합니다
- 마일스톤 이름을 바꿀 수 있습니다
- 마지막 완료 단계 앞에 새 타이머를 추가할 수 있습니다
- 도달하지 않은 중간 마일스톤을 삭제할 수 있습니다
- 진행 중에도 현재 단계 시간을 보정할 수 있습니다
- 프로필별 예열 시간, 목표 온도, 마일스톤, 프리셋 선택을 브라우저에 자동 저장합니다

자동 저장 대상은 레시피에 해당하는 설정만입니다. 진행 중인 시간, 알람 큐, 완료 여부 같은 일회성 상태는 새로고침 후 다시 불러오지 않습니다.

### 알람 처리

- 화면 전체가 깜빡입니다
- 반복 비프음
- 진동 지원 환경에서 하드웨어 진동
- 알람을 확인할 때까지 전면 모달을 유지합니다
- 여러 알람이 겹치면 큐에 쌓아 순서대로 표시
- 모달 포커스를 고정하고 `Esc`로 초기화를 취소합니다

### 완료와 연장

- 완료 시 프로필별 추가 시간을 선택할 수 있습니다
- 선택한 추가 시간은 총 시간에 반영됩니다
- 참조 줄에는 `+4분 추가됨`처럼 연장 내역을 표시합니다
- 완료 후에도 현재 리스트를 유지한 채 다시 시작할 수 있습니다
- `초기화`는 기본 리스트와 기본 시간으로 되돌립니다

### 광고와 사이트 구조

- footer에 `개인정보처리방침`, `문의` 링크를 둡니다
- `google-adsense-account` 메타, AdSense 스니펫, `ads.txt`를 포함합니다
- 배포할 때는 광고를 필요한 위치에 따로 배치할 수 있습니다

### 앱 설치와 오프라인 사용

- `manifest.webmanifest`와 `icon.svg`를 포함합니다
- 지원 브라우저에서는 홈 화면에 추가할 수 있습니다
- `service-worker.js`로 기본 파일을 캐시합니다
- `file://`로 열 때는 서비스워커 등록을 건너뜀

서비스워커는 `http://localhost` 같은 서버 환경에서만 등록됩니다. 파일을 직접 열어도 타이머 기능은 그대로 쓸 수 있습니다.

## 파일 구성

- `index.html`: 앱 UI, 스타일, 타이머 로직
- `privacy.html`: 개인정보처리방침과 광고 쿠키 고지
- `contact.html`: 문의 안내
- `manifest.webmanifest`: 설치형 앱 메타데이터
- `service-worker.js`: 기본 파일 캐시
- `icon.svg`: 앱 아이콘

## UI 방향

- 어두운 조리 도구형 팔레트를 유지합니다
- 로스팅, 만두, 라면, 계란은 선택 상태와 주요 행동 색으로 구분합니다
- 완료 상태는 초록, 위험/긴급 상태는 빨강으로 나눕니다
- 버튼과 상태 라벨은 굵은 글씨로 가독성을 높입니다
- 모바일 터치 타깃을 고려해 버튼 높이를 맞춥니다

## 기술 메모

- 빌드 도구 없이 돌아가는 단일 `index.html`
- 순수 HTML, CSS, JavaScript
- `Date.now()` 기반 시간 계산
- `requestAnimationFrame` 단일 루프에서 예열과 선택된 타이머를 관리합니다
- `Web Audio API`로 알람음을 생성합니다
- `Screen Wake Lock API`를 지원하는 환경에서는 화면을 유지합니다
- `prefers-reduced-motion`에 대응합니다
- 진행바는 `transform: scaleX()`로 갱신합니다

## 검증 메모

- 모바일 폭 `390px`에서도 가로 스크롤 없이 표시됩니다
- 저장 후 새로고침해도 선택 프로필과 마일스톤 구성이 복원되는지 확인합니다
- 완료 후 총 시간 표시와 같은 리스트 다시 시작을 확인합니다
- 라면 제품 선택 시 총 시간, 설명, 출처 링크, 기본 마일스톤 전환을 확인합니다
- 만두 제품 선택 시 냉동 / 해동 전환, 설명, 출처 링크, 기본 마일스톤 전환을 확인합니다
- 계란 제품 선택 시 냉장 / 상온 전환, 반숙 / 중간반숙 / 완숙 전환, 완료 모달 연장을 확인합니다
- 로스팅 프로필에서 원두 프리셋, 원산지, 배전, 예열 온도, 투입 온도, 향미 노트를 확인합니다
- AdSense 메타와 `ads.txt` 포함을 확인합니다
- 로컬 서버 환경에서 서비스워커 활성화를 확인합니다

## 개발용 기능

하단 `DEV` 버튼으로 알람 미리보기 패널을 열 수 있습니다. 타이머를 실제로 시작하지 않아도 현재 프로필의 마일스톤 알람과 완료 연장 팝업을 테스트할 수 있습니다.
